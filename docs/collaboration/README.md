# Git协作基线文档

更新时间: 2026-04-18

## 文档定位

本目录用于固化轻量级二手交易平台在 VibeCoding 场景下的 Git 版本控制与协作规范，目标是最少流程、最高可追溯性、最低协作风险。

## 执行原则

1. 先遵守红线，再执行开发。
2. 先保证提交可追溯，再追求迭代速度。
3. 任何超出MVP边界的改动必须先获人工确认。
