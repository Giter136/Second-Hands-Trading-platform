# 02. 轻量PRD字段与状态机

更新时间: 2026-04-18

## 1. 核心字段规则

| 功能 | 字段 | 必填 | 规则 | 说明 |
|---|---|---|---|---|
| 注册 | 手机号 | 是 | 11-20字符, 唯一 | 账号主标识 |
| 注册 | 密码 | 是 | 8-32字符, 需字母+数字 | 仅存哈希 |
| 注册 | 昵称 | 是 | 2-20字符 | 前台展示名 |
| 商品发布 | 标题 | 是 | 5-60字符 | 禁止联系方式灌入 |
| 商品发布 | 分类 | 是 | 枚举值 | 例如数码/书籍/家居 |
| 商品发布 | 成色 | 是 | 枚举值1-5 | 1最旧, 5近新 |
| 商品发布 | 价格 | 是 | >0, 最多2位小数 | 挂牌价 |
| 商品发布 | 描述 | 是 | 10-2000字符 | 说明瑕疵与使用情况 |
| 商品发布 | 图片 | 是 | 1-9张 | 至少1张封面 |
| 商品发布 | 所在城市 | 是 | 2-30字符 | 线下交易基础信息 |
| 沟通 | 商品ID | 是 | 商品须存在且可交易 | 会话与商品绑定 |
| 沟通 | 首条消息 | 是 | 2-500字符 | 空白和敏感词拦截 |
| 交易发起 | 约定价格 | 是 | >0, 最多2位小数 | 可与挂牌价不同 |
| 交易发起 | 交易地点 | 是 | 2-120字符 | 线下面交 |
| 交易发起 | 交易时间 | 否 | 合法时间格式 | 可后续补充 |

## 2. 商品状态机

待审核 -> 在售 -> 交易中 -> 已售出

待审核 -> 审核拒绝

在售 -> 已下架

交易中 -> 在售(交易取消回滚)

### 状态变更规则

1. 只有管理员可将待审核改为在售或审核拒绝。
2. 只有卖家可将在售改为已下架。
3. 交易发起后商品进入交易中，避免重复成交。
4. 交易完成后商品锁定为已售出，不可重新上架。

## 3. 交易状态机

已发起 -> 待卖家确认 -> 进行中 -> 已完成

已发起/待卖家确认/进行中 -> 已取消

### 状态变更规则

1. 买家发起交易后进入已发起。
2. 卖家确认后进入进行中。
3. 任一方在未完成前可取消，进入已取消。
4. 双方确认完成后进入已完成，并驱动商品改为已售出。

## 4. 会话状态

进行中 -> 已关闭

### 状态规则

1. 商品已售出后，会话默认只读，不可继续发起新交易意向。
2. 管理员可关闭违规会话。
